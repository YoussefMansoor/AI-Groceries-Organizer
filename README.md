# Smart Grocery Categorization System

This project categorizes groceries using manual or voice input through a terminal interface. It uses the Ollama local machine model to analyze the input and classify groceries into categories like fruits, vegetables, dairy, meat, and packaged goods.

## Features
- **Manual Input**: Users can manually enter grocery items.
- **Voice Input**: Users can speak the items, and the system transcribes the voice to text.
- **Categorization**: The system categorizes groceries into predefined categories using machine learning.
